Account Switcher

This page replaces all other login flows. Grab an AccessToken from your browser, paste it here, and the server will switch to that account.

Get an AccessToken

1
Open chatgpt.com in a browser where you are already signed in.
2
Press F12 (or Cmd+Opt+I) and open the Console tab.
3
Run the command below to copy the AccessToken.
Console command
If copy() is not supported, run:

Update account

Open chat
The token is stored on the server. Keep it private and do not share it with others.
